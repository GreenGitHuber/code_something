#### 关于多线程

C++11中加入了<thread>头文件，此头文件主要声明了std::thread线程类。

C++11的标准类std::thread对线程进行了封装。

std::thread代表了一个线程对象。应用C++11中的std::thread便于多线程程序的移值。
